本阶段将 DayAgent 前端从 Claude.ai 风格（暖暗极简）整体切换为 liminal pixel art 风格（Undertale / 独立游戏美术），并实现天气+时间驱动的动态像素背景场景。

## 第一部分：设计令牌重写

### 色彩变量对比

| 用途 | 旧值 | 新值 |
|------|------|------|
| 背景色 | `#1a1410` | `#080604` |
| 主色调 | `#d4845a`（暖琥珀） | `#e07030`（像素橙） |
| 卡片背景 | `#2a221a` | `rgba(14, 9, 5, 0.72)` |
| 卡片边框 | `rgba(255, 200, 120, 0.10)` | `rgba(224, 112, 48, 0.13)` |
| 正文色 | `#f5ede0` | `#c8b89a` |
| 暗淡文本 | `#7a6655` | `#6b5a48` |
| 导航背景 | 无独立变量 | `rgba(6, 4, 3, 0.82)` |

### 字体替换

```
旧: font-display: 'Lora', serif     →  新: 'Press Start 2P', monospace
旧: font-body: 'Inter', sans-serif  →  新: 'Courier New', 'Noto Sans SC', monospace
```

- Google Fonts CDN 只加载 Press Start 2P，不再加载 Inter/Lora
- Chinese 文本不回退到 Press Start 2P（该字体不含 CJK 字形），自动使用 monospace 回退链
- 全局 `image-rendering: pixelated` 确保图形元素像素感

### 卡片规范实现

```css
.card {
  background: rgba(14, 9, 5, 0.72);
  border: 1px solid rgba(224, 112, 48, 0.13);
  backdrop-filter: blur(8px);
}
.card::after {
  content: '';
  position: absolute;
  top: 0; left: 15%; right: 15%; height: 1px;
  background: linear-gradient(90deg, transparent, rgba(224,112,48,0.3), transparent);
}
```

关键变化：去除所有 `border-radius`，卡片全部直角，符合像素艺术无圆角特性。

## 第二部分：SceneBackground.vue 动态背景

### 架构

```
SceneBackground.vue
  ├── <canvas> 内部分辨率 320×180（16:9）
  │     └── CSS image-rendering: pixelated 放大到全屏
  ├── requestAnimationFrame 循环 @ ~10fps（像素感）
  └── <div> CRT 扫描线叠加层（CSS repeating-linear-gradient）
```

### 绘制层级（从底到顶）

| 层 | 函数 | 说明 |
|----|------|------|
| 1. 天空 | `drawSky()` | 垂直渐变，4 时段不同色：黎明橙→暗紫 / 白天灰蓝 / 傍晚暖橙 / 深夜纯黑蓝 |
| 2. 天体 | `drawCelestial()` | 夜晚：8×8 像素月亮（带陨石坑）；黎明/黄昏：8×8 像素太阳（带光晕） |
| 3. 星星 | `drawStars()` | 40 颗随机位置星星，正弦波闪烁动画，仅夜晚可见 |
| 4. 云层 | `drawClouds()` | 阴天叠加 3 团暗色像素云，缓慢漂移 |
| 5. 城市 | `drawCity()` | 12 栋不同高度的像素建筑 + 25 扇随机亮窗（暖黄光闪烁） |
| 6. 路灯 | `drawLamp()` | 2px 灯柱 + 6px 灯头 + 多层橙色光晕（脉冲呼吸） |
| 7. 粒子 | `drawRain()` / `drawSnow()` | 雨：50 个竖线粒子下落；雪：35 个白点飘落 |
| 8. 窗框 | `drawWindow()` | 左右 18px 暗色窗帘 + 顶部 8px 窗框 + 底部 10px 窗台 |
| 9. 水滴 | `drawWindowDrops()` | 15 个静态窗玻璃水滴（雨天），带拖尾 |
| 10. 像素猫 | `drawCat()` | 10×8 像素橙色猫咪，8 帧呼吸动画（上下 1px bob），眨眼效果（~2s 周期） |
| 11. 晕影 | `drawVignette()` | 径向渐变四周压暗 + 顶部额外暗角 |
| — CRT | CSS 叠加 | `repeating-linear-gradient` 2px 条纹，opacity 0.06 |

### 时段逻辑

```javascript
function getPeriod(h) {
  if (h >= 5 && h < 8) return 'dawn'    // 黎明 5:00-8:00
  if (h >= 8 && h < 17) return 'day'    // 白天 8:00-17:00
  if (h >= 17 && h < 20) return 'dusk'  // 黄昏 17:00-20:00
  return 'night'                         // 深夜 20:00-5:00（默认）
}
```

### 天气 Props

| 值 | 效果 |
|----|------|
| `sunny` | 窗外静止，无天气粒子 |
| `rain` | 雨丝粒子 + 窗玻璃水珠 |
| `snow` | 雪花飘落粒子 |
| `cloudy` | 云层叠加 + 整体降亮度 |

### 像素猫动画

```
Frame 数据：10×8 像素二维数组
- 值 0：空白
- 值 1：橙色身体 (#e07030)
- 值 2：深色轮廓/耳朵 (#6b3010)
- 值 3：眼睛 (#f8f0d0)

动画：
- 呼吸：Y 轴 ±1px 正弦波位移（周期 ~2.5s）
- 眨眼：每 120 tick 闭眼 5 tick
- 路灯橙色光晕覆盖猫身体
```

## Bug 修复记录（2026-05-19）

### Bug 1：Agent 服务超时

**现象**：Java 端报 `I/O error on POST request for "http://localhost:8000/generate-plan": Read timed out`，但 Python 端 LLM 实际成功返回结果。

**根因**：`RestTemplateConfig.java` 中 `readTimeout` 设为 45 秒，而 DeepSeek LLM 在融合多源数据生成完整规划时耗时经常超过 60 秒。

**修复**：`RestTemplateConfig.java:20` — `readTimeout` 从 `45_000` 提升到 `120_000`（2 分钟）。

### Bug 2：像素字体模糊

**现象**：Press Start 2P 字体渲染模糊，失去像素风硬朗感。

**修复（4 项）**：

| # | 文件 | 修改 |
|---|------|------|
| 1 | `global.css` 的 `*` reset | 全局加入 `-webkit-font-smoothing: none; -moz-osx-font-smoothing: unset; font-smooth: never; text-rendering: optimizeSpeed` |
| 2 | 所有 Press Start 2P 元素 | 确认已有 `image-rendering: pixelated`（App.vue 标题栏/Logo/侧边栏 nav/section-label 等） |
| 3 | `electron/main.cjs` BrowserWindow | 添加 `zoomFactor: 1.0` 到 `webPreferences` |
| 4 | `electron/main.cjs` app 启动 | 添加 `app.commandLine.appendSwitch('force-device-scale-factor', '1')` 禁止高分屏额外缩放 |

### Bug 3：页面切换卡顿

**现象**：页面切换后有白屏闪烁，内容不够丝滑。

**修复（3 项）**：

| # | 文件 | 修改 |
|---|------|------|
| 1 | `App.vue` router-view | 添加 `<keep-alive>` 包裹 `<component :is="Component">`，缓存所有页面组件避免重复渲染和重复请求 |
| 2 | `App.vue` page transition | 过渡时间从 `0.25s steps(6)` 改为 `0.15s ease`，快且平滑 |
| 3 | 路由懒加载 | 已确认所有路由使用 `() => import()` 动态导入（无需修改） |
